### خلاصه
یک فروشگاه آنلاین واکنش‌گرا (Responsive) ساخته‌شده با React و Tailwind CSS که تجربه کاربری ساده، سریع و امن برای خرید آنلاین فراهم می‌کند. این پروژه شامل ثبت‌نام و ورود کاربر، مدیریت سبد خرید، نمایش محصولات از طریق API، و اعتبارسنجی فرم‌ها با استفاده از Formik و Yup است. داده‌های محلی مانند سبد خرید و وضعیت ورود کاربر از طریق Local Storage نگهداری می‌شوند. مسیریابی با `React Router` و مدیریت وضعیت سراسری با `Context API` پیاده‌سازی شده‌اند.

### ویژگی‌ها
- طراحی کاملاً ریسپانسیو با Tailwind CSS (سازگار با موبایل، تبلت و دسکتاپ)
- نمایش لیست محصولات پویا از یک API
- صفحات محصول و صفحه جزئیات محصول
- سیستم ثبت‌نام و ورود کاربر
  - فرم‌های مدیریت شده با Formik
  - اعتبارسنجی ورودی‌ها با Yup
- سبد خرید:
  - افزودن/حذف محصولات
  - نمایش مجموع قیمت و تعداد اقلام
  - نگهداری سبد خرید در Local Storage برای حفظ وضعیت بین رفرش‌ها
- مدیریت وضعیت سراسری با `Context API` (برای اشتراک‌گذاری اطلاعات کاربر و سبد خرید)
- مسیریابی صفحات با `React Router`
- پیاده‌سازی بهترین شیوه‌ها برای تجربه کاربری و بهینه‌سازی سرعت

### تکنولوژی‌ها و کتابخانه‌ها
- React
- Tailwind CSS
- React Router
- Context API
- Formik
- Yup
- Local Storage
- Axios (یا fetch) برای درخواست‌های HTTP
